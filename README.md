## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

you can get a free pocketbase database using fly.io and by following [this](https://github.com/pocketbase/pocketbase/discussions/537) guide
